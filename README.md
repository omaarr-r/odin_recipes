# odin_recipes
Description
This project is a basic recipe website built as part of The Odin Project's Foundations Course. The primary goal of this assignment was to demonstrate a solid understanding of fundamental HTML concepts, including document structure, linking, and asset management.

Features & Skills Demonstrated
In this project, I successfully implemented:

HTML Structure: Organizing content using proper headings, paragraphs, and semantic tags.

Navigation: Linking multiple HTML pages together using relative paths.

Media Integration: Implementing images with appropriate alternative text for accessibility.

Lists: Utilizing both ordered (<ol>) and unordered (<ul>) lists to display ingredients and cooking steps.

Version Control: Using Git and GitHub to manage the development process and host the repository.

Project Structure
The repository is organized to ensure clean file management:

index.html: The main landing page containing the list of recipes.

/recipes/: A dedicated directory containing individual HTML files for each recipe.

/images/: A folder storing the visual assets used across the site.

Reflection
This project served as a crucial first step in my web development journey. It allowed me to move from theoretical knowledge to practical execution, specifically focusing on how to structure information effectively before moving on to styling with CSS.
